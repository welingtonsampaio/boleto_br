# BoletoBr

Sistema para a emissão de boletos bancários utilizando a linguagem Ruby

## Instalação

Adicione esta linha no seu arquivo Gemfile de sua aplicação:

    gem 'boleto_br'

Depois execute no terminal:

    $ bundle install

Ou instale manualmente:

    $ gem install boleto_br
